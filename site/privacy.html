Privacy Policy

We care about your privacy and do not collect any user data. In order to provide the best possible Nitro experience we collect anonymous analytics so we can measure traffic.

Information we Collect

We do not collect any user data but we collect anonymous analytics when Nitro is synced.

  • OS (e.g. windows)
  • Language (e.g english)
  • Nitro version (e.g. 1.2)
  • Nitro backend (the service you use to store your Nitro information e.g Dropbox, Ubuntu One)
  • User ID (e.g. 1235612756)

How we use this information

We collect anonymous analytics so we can measure traffic and provide the best possible experience.

Our Nitro sync server does not store data. Your data is in the hands of the service you choose. Before your data can be synced to the storage backend of your choosing, it needs to go through our server to be merged. Our server prevents creation of “conflicted copies”. If you wish to run this server yourself, the software is avaliable at http://github.com/stayradiated/Nitro-Sync

Data Security

While we do not store user data, all traffic to our server is encrypted. Authorization between the computer and the storage backend is done securely and we only have access to the information that the Application Programming Interface (API) provides us with.

External Services

Data is stored on the external service of your choice. Please refer to their own privacy policies.

  • Dropbox
  • Ubuntu One

Changes to this policy

This Privacy Policy may change from time to time. If we make a change to the privacy policy that we believe materially reduces your rights, we will provide you with notice (for example, by email).
